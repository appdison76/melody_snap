# MelodySnap — Play 스토어 버전

이 프로젝트는 **Google Play 스토어 출시용** 앱입니다.

## 포함 기능 (탭 3개)
- **음악 찾기** — 음악 인식
- **검색** — YouTube 검색
- **찜하기** — 즐겨찾기

## 제외된 기능
- **저장** 탭 (URL 붙여넣기 / 다운로드)
- **내 파일** 탭 (다운로드한 파일 목록)

다운로드 관련 버튼을 누르면 "Play 스토어 버전에서는 다운로드 기능을 제공하지 않습니다" 안내가 표시됩니다.

## 빌드
- `npm install` 후 `npx expo run:android` 또는 기존 빌드 스크립트 사용
- 전체 버전(저장/내파일 포함)은 `youtube_down/app` 프로젝트에서 빌드

## 원본
- 소스 복사 출처: `youtube_down/app`
- 백엔드 서버는 youtube_down의 server를 그대로 사용 (검색, 자동완성, 음악 인식 API)
